# Computer Vision Mini Projects (C++ / OpenCV)

This repository contains several small projects related to computer vision and image processing.

Each project was developed for understanding fundamental computer vision techniques.

The projects are implemented mainly in **C++ using OpenCV**.

---

## Repository Structure

```text
.
├── chroma_correction/hdr/
├── edges_optical_flow/
├── hdr/
└── vision_3d/

```
---

## Projects

### HDR
High Dynamic Range image generation from multiple exposures.

Main topics:
- exposure fusion
- radiance maps
- tone mapping

---

### Chroma & Color Correction
Image color processing techniques including chroma keying and color correction.

Main topics:
- chroma key (green screen)
- color space transformations
- image correction

---

### Edges & Optical Flow
Detection of edges and motion estimation between frames.

Main topics:
- edge detection
- gradient operators
- optical flow

---

### 3D Vision
Basic 3D reconstruction techniques using multiple images.

Main topics:
- camera geometry
- depth estimation
- stereo vision

---

Each folder contains its own **README with details about the specific project**.
